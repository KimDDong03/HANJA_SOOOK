수동 획순 제작 샘플: 家

아래 path는 외부 획순 데이터를 복사하지 않고 의사결정용으로 직접 그린 샘플입니다. 실제 MVP에서는 같은 형식의 SVG path를 `StrokeAsset.svgPaths`에 저장해서 guided writing에 사용합니다.

1. 정밀 글자형

글자 모양은 서체 렌더링으로 고정하고, 파란 점선은 수동 제작한 획순 중심선입니다. 실제 앱에서는 점선은 숨기고 채점/획순 진행에만 씁니다.

2. 획순 번호

수동 제작물에는 번호와 시작점 힌트를 같이 저장할 수 있습니다. 아이 대상 UI에서는 번호 대신 현재 획만 강조하는 편이 더 깔끔합니다.

3. 앱 화면 느낌
