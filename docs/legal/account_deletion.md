# 한자쏘옥 계정 및 데이터 삭제 요청

공고일: 2026-06-11
시행일: 2026-06-11

한자쏘옥 앱에서 생성된 계정과 관련 학습 데이터 삭제를 요청할 수 있습니다.

## 삭제 대상

- 앱 내부 사용자 계정
- 학생 이름 또는 닉네임
- 학교명, 표준학교코드, 학년
- 학습 진도, 퀴즈 결과, 게임 결과, 획쓰기 연습 결과
- 앱 설정 및 알림 설정

## 요청 방법

Google Play 스토어 등록정보에 표시된 개발자 연락처로 아래 정보를 포함해 요청해 주세요.

- 앱 이름: 한자쏘옥
- 요청 유형: 계정 및 데이터 삭제
- 앱에서 사용한 이름 또는 닉네임
- 학교명과 학년
- 보호자 또는 법정대리인 요청 여부

요청이 접수되면 본인 확인과 삭제 범위 확인 후 계정과 관련 데이터를 삭제하거나 더 이상 개인을 식별할 수 없도록 처리합니다.

## 처리 안내

삭제가 완료되면 해당 계정의 학습 진도와 결과는 복구할 수 없습니다. 법령상 보관이 필요한 정보가 있는 경우 해당 기간 동안 보관한 뒤 삭제합니다.
