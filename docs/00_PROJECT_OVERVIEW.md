# 00. Project Overview

## 목적

한자쏘옥 1차 MVP는 학교/교사/학부모에게 앱의 학습 체험을 보여주기 위한 Flutter 기반 시연 앱이다.
첫 버전은 실서비스 전체 구축이 아니라, 아래 학습 루프가 끊기지 않고 동작하는 것을 목표로 한다.

```txt
학교 검색 → 이름/학년 입력 → 학생 홈 → 한자 카드 → 따라쓰기 → 결과 → 퀴즈/게임 → 결과 → 성장 화면
```

## 장기 방향

1차는 MVP지만, 코드 구조는 실서비스 확장을 고려한다.
따라서 Supabase, Repository, Domain Service, Drift cache, StrokeEngine adapter 구조를 처음부터 만든다.

## 1차 MVP 성공 기준

- Android 테스트 단말에서 설치 및 실행 가능
- 학교 검색/선택 가능
- 학생 프로필 생성 가능
- 홈에서 한자 학습으로 이동 가능
- 한자 카드에서 정보 확인 가능
- 손가락으로 따라쓰기 가능
- 퀴즈/게임 10문항 흐름 가능
- 결과 화면에서 점수/정답률/XP 표시 가능
- 성장 화면에서 레벨/배지 표시 가능
- Missing data가 있어도 앱이 죽지 않고 fallback 처리

## 비목표

- 실서비스 운영용 전체 관리자
- 모든 교사/학부모 기능
- AI 채점
- 앱스토어 심사 완료 보장
- 전체 콘텐츠 검수 완료
